# Inventory & Sales Management SQL Project

This repository contains an end-to-end SQL project for Inventory and Sales Management.
It includes schema definitions, sample data, triggers, stored procedures and analytical queries.

## How to use
1. Run `02_Database_Schema/create_tables.sql`
2. Run `03_Sample_Data/insert_data.sql`
3. Run `04_Triggers/stock_update_trigger.sql`
4. Run `05_Stored_Procedures/sales_report_procedure.sql`
5. Explore queries in `06_Queries/analytical_queries.sql`

## Folder structure
- 02_Database_Schema/
- 03_Sample_Data/
- 04_Triggers/
- 05_Stored_Procedures/
- 06_Queries/

## Notes
- Files are written to be compatible with MySQL/MariaDB. Minor syntax tweaks may be needed for PostgreSQL or SQL Server.
- Always backup your data before running DDL/DML on an existing database.
